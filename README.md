# 除螨仪覆盖率测试软件

基于OpenCV的除螨仪覆盖率测试软件，通过手机录像分析除螨仪的清洁覆盖率。

## 功能特点

- ✅ 床铺区域标定（4点透视变换）
- ✅ 蓝色标记追踪（除螨仪上的蓝色图纸）
- ✅ 覆盖率计算（考虑主刷口大小）
- ✅ 热力图生成
- ✅ 轨迹可视化
- ✅ 统计报告生成

## 系统要求

- Python 3.7+
- OpenCV 4.8+
- NumPy 1.24+
- Matplotlib 3.7+

## 安装

```bash
# 克隆或下载项目
cd mite_coverage

# 安装依赖
pip install -r requirements.txt
```

## 快速开始

### 1. 准备材料

- 手机支架（固定手机拍摄）
- 蓝色纸张（贴在除螨仪上作为标记）
- 测试录像（手机拍摄的除螨仪工作视频，包含床铺全景）

### 2. 标定床铺区域

**方式一：从视频第一帧标定（推荐）**

1. 将测试录像放入 `videos/` 目录
2. 运行程序：
   ```bash
   python main.py
   ```
3. 选择菜单 `1. 标定床铺区域`
4. 选择 `1. 从视频第一帧标定`
5. 选择对应的视频文件
6. 在视频第一帧上依次点击床铺的4个角点（顺时针或逆时针）
7. 按 `S` 键保存配置
8. 按 `Q` 键退出

**方式二：从图片标定**

1. 将床铺照片放入 `images/` 目录，命名为 `test.jpg`
2. 运行程序：
   ```bash
   python main.py
   ```
3. 选择菜单 `1. 标定床铺区域`
4. 选择 `2. 从图片标定`
5. 在图片上依次点击床铺的4个角点（顺时针或逆时针）
6. 按 `S` 键保存配置
7. 按 `Q` 键退出

### 3. 准备视频

1. 将测试录像放入 `videos/` 目录
2. 支持的格式：`.mp4`, `.avi`, `.mov`
3. 确保视频中包含蓝色标记（贴在除螨仪上的蓝色纸张）

### 4. 分析录像

1. 运行程序：
   ```bash
   python main.py
   ```
2. 选择菜单 `2. 分析录像`
3. 选择要分析的视频文件
4. 等待分析完成
5. 查看结果

### 5. 查看结果

分析完成后，结果将保存在 `outputs/` 目录：

- `coverage_result.jpg` - 覆盖结果图（绿色=已覆盖，红色=未覆盖）
- `heatmap.jpg` - 热力图（颜色深浅表示覆盖次数）
- `report.jpg` - 综合报告
- `summary.jpg` - 摘要图像
- `trajectory.csv` - 轨迹数据
- `tracking_result.mp4` - 追踪视频
- `coverage_curve.png` - 覆盖率曲线

## 配置参数

### 颜色检测参数

编辑 `config.json` 文件：

```json
{
  "tracker": {
    "blue_lower_h": 90,
    "blue_lower_s": 50,
    "blue_lower_v": 50,
    "blue_upper_h": 140,
    "blue_upper_s": 255,
    "blue_upper_v": 255,
    "min_area": 50,
    "morph_kernel_size": 5
  },
  "coverage": {
    "brush_radius": 20,
    "grid_size": 10
  }
}
```

### 参数说明

#### 追踪器参数
- `blue_lower_h/s/v`: 蓝色HSV范围下限
- `blue_upper_h/s/v`: 蓝色HSV范围上限
- `min_area`: 最小检测区域（像素）
- `morph_kernel_size`: 形态学操作核大小

#### 覆盖率参数
- `brush_width`: 主刷口宽度（像素），根据除螨仪实际宽度和视频分辨率调整
- `brush_height`: 主刷口高度（像素），根据除螨仪实际高度和视频分辨率调整
- `grid_size`: 网格大小（像素），用于计算覆盖率

## 使用技巧

### 1. 蓝色标记建议

- 使用鲜艳的蓝色纸张
- 确保标记在视频中清晰可见
- 避免反光和阴影

### 2. 拍摄建议

- 使用手机支架固定，避免抖动
- 确保床铺在画面中完整显示
- 光线均匀，避免强光直射
- 视频分辨率建议 720p 或 1080p

### 3. 参数调整

如果追踪效果不好，可以调整：
- 蓝色HSV范围（使用颜色选择器工具）
- 最小区域大小（过滤噪点）
- 形态学核大小（去噪效果）

## 常见问题

### Q: 蓝色标记追踪失败？

A: 检查以下几点：
1. 调整蓝色HSV范围
2. 确保标记颜色鲜艳
3. 检查光线条件
4. 调整最小区域大小

### Q: 覆盖率计算不准确？

A: 可以调整：
1. 主刷口半径（根据实际除螨仪大小）
2. 网格大小
3. 确保床铺区域标定准确

### Q: 视频处理速度慢？

A: 这是正常的，软件会逐帧处理。可以：
1. 降低视频分辨率
2. 减少视频长度
3. 跳过一些帧（修改代码）

## 技术实现

- **颜色检测**: HSV颜色空间 + 形态学操作
- **透视变换**: OpenCV getPerspectiveTransform
- **覆盖率计算**: 网格化 + 圆形主刷口模拟
- **可视化**: OpenCV + Matplotlib
