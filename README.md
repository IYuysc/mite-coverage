# 除螨仪覆盖率测试软件

基于OpenCV的除螨仪覆盖率测试软件，通过手机录像分析除螨仪的清洁覆盖率。

## 功能特点

- ✅ 床铺区域标定（4点透视变换）
- ✅ 双色标记联合追踪（左侧翠绿色贴纸，右侧蓝色贴纸，左右并排）

- ✅ 斜视角高度视差投影线性拉回补偿
- ✅ 覆盖率计算（根据实际刷头长宽和网格大小自动转换）
- ✅ 热力图生成与轨迹可视化
- ✅ 统计报告生成（覆盖率趋势曲线图与扫掠图拼接）
- ✅ 全新现代化 PyQt5 图形界面，告别命令行黑框
- ✅ 参数一键可视化实时配置（包含噪点、高度视差参数调整等）

## 系统要求

- Python 3.7+
- OpenCV 4.8+
- NumPy 1.24+
- Matplotlib 3.7+
- PyQt5 5.15+

## 安装

```bash
# 克隆或下载项目
cd mite_coverage

# 安装依赖
pip install -r requirements.txt
```

## 无需环境直接运行（执行文件方式）

如果您不想安装 Python 及相关依赖，可以直接使用打包好的 `MiteCoverage.exe` 执行文件运行软件：

1. 将生成的 `MiteCoverage.exe`（通常在 `dist` 目录下）复制到包含 `videos/` 和 `images/` 文件夹的工作目录中。
2. 确保同一目录下有您的 `videos/` (存放测试视频) 和 `images/` (如果有标定图片) 文件夹。
3. 双击运行 `MiteCoverage.exe` 即可直接打开软件，终端界面和操作方法与源代码运行完全一致。

## 快速开始

### 1. 准备材料

- 手机支架（固定手机拍摄）
- 双色标记贴纸（左侧贴**鲜艳翠绿色纸张**，右侧贴**鲜艳蓝色纸张**，水平并排固定在除螨仪顶部）
- 测试录像（手机拍摄的除螨仪工作视频，包含床铺全景）

### 2. 标定床铺区域

1. 将测试录像放入 `videos/` 目录
2. 运行程序（双击 `MiteCoverage.exe` 或运行 `python main.py`，将弹出图形界面）
3. 点击主界面上的 **"标定床铺区域"** 按钮
4. 在弹出的对话框中选择对应的视频文件
5. 在弹出的视频第一帧上，依次点击床铺的4个角点（顺时针或逆时针）
6. 按 `S` 键保存配置
7. 返回主界面的 **"参数配置"** 选项卡，输入床铺实际长宽尺寸（厘米）并点击“保存配置”生效

### 3. 准备视频

1. 将测试录像放入 `videos/` 目录
2. 支持的格式：`.mp4`, `.avi`, `.mov`
3. 确保视频中包含双色标记（左侧翠绿色，右侧蓝色）

### 4. 分析录像

1. 运行程序：
   ```bash
   python main.py
   ```
2. 点击主界面上的 **"计算覆盖率"** 按钮
3. 在系统对话框中选择要分析的视频文件
4. 可以在界面底部的文本框实时查看追踪运算进度与日志
5. 分析完成后（或按追踪窗口右上角的 `X` 提前结束时），系统会自动弹出覆盖率报告图

### 5. 查看结果

分析完成后，结果将保存在 `outputs/` 目录：

- `heatmap.jpg` - 半透明覆盖热力图（透视变换还原叠加在视频第一帧视角上，直观展示清扫重合度）
- `report.jpg` - 综合分析报告图（左侧为原始视角下的淡蓝色清扫路线图，右侧为覆盖率随时间的变化趋势曲线）
- `tracking_result.mp4` - 追踪视频（带实时追踪定位框与累计扫掠路网的可视化视频）

**注意**：如果标定时输入了床铺实际尺寸，程序会显示实际覆盖面积（cm² 和 m²）。

## 配置参数

### 颜色检测与算法参数

直接在软件界面的 **"参数配置"** 标签页中直观修改，或手动编辑 `config.json` 文件：

```json
{
  "tracker": {
    "blue_lower_h": 100,
    "blue_lower_s": 90,
    "blue_lower_v": 50,
    "blue_upper_h": 130,
    "blue_upper_s": 255,
    "blue_upper_v": 250,
    "green_lower_h": 30,
    "green_lower_s": 35,
    "green_lower_v": 35,
    "green_upper_h": 55,
    "green_upper_s": 255,
    "green_upper_v": 255,
    "min_area": 50,
    "morph_kernel_size": 5,
    "sticker_layout": "side_by_side",
    "parallax_height_cm": 8.0,

  },
  "coverage": {
    "brush_width": 30,
    "brush_height": 15,
    "grid_size": 10,
    "output_dir": "outputs",
    "real_brush_width_cm": 11.4,
    "real_brush_height_cm": 2.3,
    "real_remover_width_cm": 25.0,
    "real_remover_height_cm": 21.0
  }
}
```

### 参数说明

#### 追踪器参数
- `blue_lower_h/s/v`: 蓝色 HSV 范围下限
- `blue_upper_h/s/v`: 蓝色 HSV 范围上限
- `green_lower_h/s/v`: 翠绿色 HSV 范围下限
- `green_upper_h/s/v`: 翠绿色 HSV 范围上限
- `min_area`: 最小检测区域（像素）
- `morph_kernel_size`: 形态学操作核大小
- `sticker_layout`: 贴纸排布方式（固定为并排模式 `"side_by_side"`）
- `parallax_height_cm`: 贴纸表面距离床面的垂直高度（单位：cm），用于消除斜视角下的视差偏差


#### 覆盖率参数
- `real_brush_width_cm`: 实际刷头宽度（cm）
- `real_brush_height_cm`: 实际刷头长度（cm）
- `real_remover_width_cm`: 实际除螨仪宽度（cm）
- `real_remover_height_cm`: 实际除螨仪长度（cm）
- `grid_size`: 网格划分大小（像素），用于计算覆盖率的离散单元大小

## 使用技巧

### 1. 标记建议

- 贴纸排布必须为并排模式（左侧贴鲜艳翠绿，右侧贴鲜艳蓝色）
- 确保标记在视频中清晰可见，避免手部或电线大面积遮挡
- 尽量避免过强强光产生的局部大镜面反射

### 2. 拍摄建议

- 使用手机支架固定，避免抖动
- 确保床铺在画面中完整显示
- 光线均匀，避免强光直射
- 视频分辨率建议 720p 或 1080p

### 3. 参数调整

如果追踪效果不好，可以调整：
- 蓝/绿两色 HSV 范围（利用配置菜单实时调试）
- 最小检测区域大小 `min_area`
- 形态学滤波核大小 `morph_kernel_size`

## 工作流程

```
1. 准备视频
   └── 将测试录像放入 videos/ 目录

2. 标定床铺区域（从视频第一帧）
   └── videos/*.mp4 → 提取第一帧 → 选择4个角点 → config.json

3. 分析录像
   └── videos/*.mp4 → 追踪双色标记 → 轨迹数据

4. 坐标转换与视差修正
   └── 视频坐标 → 透视变换 → 线性视差补偿 → 床铺物理坐标

5. 计算覆盖率
   └── 轨迹点 + 旋转矩形主刷口模拟 + 局部网格测试 → 覆盖率统计

6. 生成报告
   └── 逆向透视变换回原始画面第一帧视角 → 叠加绘制清扫网/热力图 → 拼接趋势曲线 → outputs/
```

## 常见问题

### Q: 双色标记追踪失败？

A: 检查以下几点：
1. 分别微调绿/蓝两色 HSV 范围，特别是翠绿色（Hue 50-85）和蓝色（Hue 100-130）。
2. 确保贴纸颜色鲜艳、反光较小。
3. 检查光线条件，避免过强日光直射产生大面积死白反光。
4. 调整 `min_area` 最小连通域检测面积（若贴纸在视频中像素占比小，可调低此值）。

### Q: 覆盖率计算不准确？

A: 可以调整：
1. 配置好除螨仪的实际物理长宽（`real_remover_width_cm`, `real_remover_height_cm`）与主刷口长宽（`real_brush_width_cm`, `real_brush_height_cm`）。
2. 根据镜头斜射角度，准确量取贴纸表面到床单的高度并填入 `parallax_height_cm` 视差修正中。
3. 标定床铺（4角点）时力求高精度，特别是床头远端两个角点不要选错。

### Q: 视频处理速度慢？

A: 由于逐帧检测和生成高精度扫掠掩码，处理耗时属正常现象。可以：
1. 适当降低原视频分辨率（如使用 720p 视频）。
2. 在 `4. 配置参数` 中将 `网格大小` 从 10 像素微调大（网格越大，离散化网格测试计算开销越低）。

## 技术实现

- **双色追踪**: HSV 颜色分割 + 二值形态学滤波

- **空间投影**: 双向 Perspective Homography 透视矩阵映射
- **高度补偿**: 基于视差插值的垂直方向像素拉回校正
- **覆盖计算**: 旋转矩形扫掠网格化离散测试（`cv2.pointPolygonTest`） + 局部包围盒增量热力图优化

## 注意事项

1. 首次使用前，必须先在主界面点击 **"标定床铺区域"** 进行标定。
2. 贴纸排布必须为并排模式（左侧贴鲜艳翠绿，右侧贴鲜艳蓝色，贴纸中心到吸尘面高度一致）。
3. 所有的除螨仪和主刷口实际尺寸需仔细用钢尺量取，并在“参数配置”面板中填入，以确保覆盖面积厘米级精度。
